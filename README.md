# prismagram

intagram clone with Express + Prisma + React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like or Unlike a photo
- [x] Comment on a photo
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow User
- [ ] See user profile
- [ ] See the full photo
- [ ] Edit my profile
- [ ] Upload a photo
- [ ] Edit or Delete the photo
- [ ] See the feed
